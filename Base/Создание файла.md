# Создание файла
Создал фалй → Статус "Untracked"
Git add → Статус "staged"
Git commit Статус "Commited" (зафиксированный) (Сделать снимок)

---

### Изменили файл
Создал фалй → Статус "Modified"
Git add → Статус "staged"
Git commit Статус "Commited" (зафиксированный) (Сделать снимок)
